Constant across all four

Same data layer, same brand, same component vocabulary. Only layout, navigation and density change — never the meaning.

↔
Context of use drives it

Phones optimise for quick actions in motion; large displays optimise for dense analysis at a desk. The breakpoints encode that intent.
